How to use this site

  • Open a blueprint below, then copy each field into the matching builder fields.
  • Knowledge — use the text as a checklist of what to upload in Gemini’s Knowledge step (this site does not host files).
  • Personalization — paste example / helper prompts so users get starter questions that fit this agent after it is created.
  • Other fields map directly to the agent builder (name, description, instructions, model).
